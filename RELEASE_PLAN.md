# Release Plan

## Release Target

- Tag: `v4.0.0`
- Title: `MSlide² Framework 4.0.0 Starter`
- Asset: `MSlide2-Framework/_release/MSlide2-Framework-4.0.0-starter-clean.zip`

## Release Scope

This release includes MSlide² only.

MSlide² is the current public MVP starter framework. It is available now and is the only framework included in the `v4.0.0` release asset.

## MSlide³ Status

MSlide³ is planned for a future release.

`MSlide3-Framework/` is currently a release-readiness shell and placeholder for the future Content State Navigation Framework. It is not part of the current public starter release, and no MSlide³ production runtime or release asset should be published for `v4.0.0`.

## Release Rules

- Do not rebuild the starter ZIP during this preflight pass.
- Do not create an MSlide³ release asset.
- Do not modify framework runtime behavior.
- Do not modify `mslide2/4/`.
- Keep internal planning reports under `docs/internal/`.
