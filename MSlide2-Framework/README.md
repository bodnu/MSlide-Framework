# MSlide² Framework

Structured Content Exploration Framework

<!-- Repository screenshots will be added before public release -->

<!-- Repository header image placeholder -->

MSlide² is a free starter framework and reference implementation for structured content exploration.

Built on Reveal.js and Bulma, it provides a lightweight foundation for interactive publications, catalogs, brochures, documentation, and presentation-like experiences.

## Features

- Structured content exploration
- Horizontal navigation
- Vertical stacks
- Desktop-first layouts
- Reveal.js foundation
- Bulma integration
- Lightweight starter package
- Localhost-first development flow

## Who Is This For?

MSlide² is intended for creators and teams building structured content experiences such as:

- Catalogs
- Brochures
- Documentation
- Reports
- Interactive publications

The starter provides the foundation while allowing projects to evolve in their own direction.

## What MSlide² Is Not

MSlide² is not:

- A Bootstrap replacement
- A Reveal.js competitor
- A general-purpose UI framework

It is a starter framework and reference pattern for structured content exploration.

## Philosophy

MSlide² focuses on structured content exploration rather than traditional long pages or linear slide decks.

The starter framework emphasizes:

- Simplicity
- Desktop-first experiences
- Horizontal sections
- Vertical stacks
- Progressive enhancement

The goal is to provide a lightweight foundation rather than a complete application framework.

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd MSlide2-Framework
```

## Quick Start

Serve the starter locally:

```bash
python3 -m http.server 8080
```

Open:

```text
http://localhost:8080
```

Do not open the starter with `file://`. Browser security rules can block assets, plugins, and runtime behavior when the project is not served through localhost.
