# Security Policy

## Supported Versions

MSlide Framework is currently in its first public MVP release phase.

| Version | Supported |
|--------|-----------|
| v4.0.0 | Yes |

## Reporting a Vulnerability

If you discover a security issue in MSlide Framework, please do not open a public issue.

Instead, report it privately by contacting:

mslide@bod.nu

Please include:

- A short description of the issue
- Steps to reproduce it
- The affected framework folder, if known
- Any relevant browser, operating system, or dependency details

## Scope

This policy applies to the public MSlide Framework repository, including:

- MSlide² Framework starter
- Future MSlide³ Framework releases
- Repository documentation and release assets

It does not apply to third-party services, external websites, or dependencies outside the repository.

## Dependency Security

MSlide Framework uses third-party open web technologies, including Reveal.js and Bulma.

Security issues in third-party dependencies should also be reported to the maintainers of those projects.

## Response

We will review valid vulnerability reports and respond as soon as reasonably possible.

For the current MVP phase, response times may vary.
